# Replace empty by na

Replace empty by na

## Usage

``` r
empty_as_na(vec)
```

## Arguments

- vec:

  Vector of values

## Value

Vector

## Examples

``` r
airquality[which(is.na(airquality$Ozone)), "Ozone"] <- ""
empty_as_na(airquality$Ozone)
```
