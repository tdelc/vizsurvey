# Preparation of a survey

Preparation of a survey

## Usage

``` r
prepa_survey(folder_path, ...)
```

## Arguments

- folder_path:

  folder of survey

- ...:

  argument to pass to folder_to_df (example : file_pattern)

## Value

NULL (creation of rds)

``` r
if (FALSE) { # \dontrun{
```
